# LA City Permitting System - Portal Mockup

A modern, clean wireframe mockup for the Los Angeles City Permitting System portal.

## Features

- **Modern Design**: Clean header with navigation menu, search, notifications, and user profile
- **Hero Section**: Construction-themed image grid with overlay text
- **Action Cards**: Four main action cards with gradients and icons:
  - New Permit
  - My Permits
  - Pay Fee
  - Permit Guide
- **Single Page Application**: Navigation between pages without reloading
- **Responsive Design**: Fully responsive layout for desktop, tablet, and mobile
- **User Interactions**: 
  - Clickable navigation menu
  - User profile dropdown with options
  - Notification bell with badge
  - Interactive cards

## Files

- `portal-simple.html` - Main portal mockup with all functionality
- `logo.png` - Los Angeles Department of Building and Safety logo
- `portal-wireframe.html` - Alternative wireframe version
- `portal.html` - Original Salesforce portal HTML

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- Montserrat Font (Google Fonts)
- Heroicons (SVG icons)
- Unsplash Images

## Usage

Simply open `portal-simple.html` in a web browser to view the mockup.

### Navigation

- **Home**: Returns to the homepage
- **New Permit**: Opens new permit application page
- **My Permits**: Opens existing permits management page
- **User Profile Dropdown**: Click on "John Doe" to access:
  - User Profile
  - Join Company
  - Log Out

## Design Specifications

### Colors
- Primary Blue: `#005587`
- Background: `#f4f6f9`
- Card Gradients: Navy, Terra Cotta, Steel Blue, Forest Green

### Typography
- Font Family: Montserrat
- Hero Text: 3.75rem
- Card Headings: 1.375rem

### Layout
- Max Width: 1200px
- Header: Sticky position
- Cards: Auto-fit grid (4 columns on desktop, 2 on tablet, 1 on mobile)
